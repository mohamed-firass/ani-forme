# Ani-Forme

Ani-Forme est une application de gestion en C# pour clinique vétérinaire
Cette application a été développé au cours de la formation ENI 2015.

Elle est entièrement codé en C# est fonctionne avec

  - Un serveur SQLServer
  - Un fichier txt contenant vos fournisseurs
  - Et un peu de magie

Les autheurs sont :
> Erwan Lemouroux & Cordelier Tom

### Technique

* [DapperNet]
* [NET Framework]
* [SQL Server]

### Installation

Vous devez juste configurer les differents params dans la app.config généré
et installer les différents fichiers SQL sur votre base de donnée
